# DeepGuard AI

## Overview

DeepGuard AI is an AI-powered Deepfake Detection Platform that helps users determine whether an uploaded image or video is authentic or AI-generated.

## Features

- User Authentication
- Image Deepfake Detection
- Video Deepfake Detection
- Confidence Score
- Scan History
- Download Reports
- Responsive Design

## Tech Stack

### Frontend
- React
- Tailwind CSS

### Backend
- Node.js
- Express.js

### Database
- MongoDB

### AI
- Python
- FastAPI
- PyTorch
- OpenCV
